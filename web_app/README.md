# Tender Web App (React SPA)

React-приложение для просмотра отчётов по тендерам.
Берёт данные из `../storage/tenders/jsonl/all_tenders_data_*.jsonl` и показывает их в виде вкладок «Отчет от <дата>».

## Стек

- **Фронт:** React 18 (через CDN UMD) + JSX компилируется в браузере (Babel standalone) — без npm/Node
- **Бэк:** Flask (отдаёт SPA + JSON API)

## Возможности

- Вкладки по каждому jsonl-отчёту (имя — дата и время рассылки)
- Фильтры: «Все / Интересные / Не интересные / Подходящие (Глубокий анализ)»
- Фильтр по категории (ИИ)
- Переключатель «Основные / Все поля»
- Сортировка по любому столбцу (с учётом цены и русских дат)
- Поиск с подсветкой совпадений: номер тендера, ключевые слова в названии, заказчик
- Подсветка строк (интересные / не подходит)
- Гиперссылки на номера тендеров

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python app.py
```

Или двойным щелчком по `run.bat` (Windows).

Открыть в браузере: http://127.0.0.1:5000/

> При первом запуске нужен интернет (чтобы скачать React и Babel с CDN).
> После этого браузер закеширует библиотеки.

## Структура

```
web_app/
├── app.py            # Flask backend (API + раздача SPA)
├── requirements.txt
├── run.bat
└── static/
    ├── index.html    # React shell (загружает React + Babel + app.jsx)
    ├── app.jsx       # React-компоненты: App, TabBar, FiltersBar, TenderTable
    └── style.css
```

## API

- `GET /api/reports` — список доступных отчётов
- `GET /api/reports/<report_id>` — записи отчёта (id = имя файла без `.jsonl`)
